# Meraki Network Setup

## 📍 Locations
- Pan India Deployment (Delhi, Mumbai, Pune and other locations)

## 🔧 Devices Used
- MX (Security Appliance)
- MS (Switches)
- MR (Access Points)

## ⚙️ Configuration Details
- VLANs configured for network segmentation
- Inter-VLAN routing enabled
- DHCP configured for end devices
- Trunk and access ports configured on switches
- Wireless SSIDs configured with access control

## 🔐 Security Configuration (Meraki MX)
- Layer 3 & Layer 7 Firewall rules applied
- IDS/IPS enabled (Intrusion Detection & Prevention)
- Advanced Malware Protection (AMP) enabled
- Content filtering configured
- VPN setup for secure site-to-site connectivity

## 🛠️ Troubleshooting
- Checked uplink status and WAN connectivity
- Verified VLAN tagging and switch configuration
- Monitored alerts via Zoho 24x7
- Performed root cause analysis for network issues
