# Common Network Issues & Fixes

## 🔴 1. Device Unreachable
- Checked device online status in Meraki dashboard
- Verified WAN connectivity
- Rebooted device remotely if required

## 🟠 2. VLAN Issue
- Checked VLAN ID mismatch
- Verified trunk and access port configuration
- Ensured proper tagging

## 🟡 3. Slow Network
- Checked bandwidth utilization
- Verified uplink speed
- Analyzed traffic via Meraki dashboard

## 🔵 4. Alert Handling (Zoho 24x7)
- Identified alert type
- Performed root cause analysis
- Resolved issue within SLA
- Updated ticket in Zoho tool

## 🟢 5. VPN Issues
- Checked tunnel status
- Verified peer configuration
- Restarted VPN service if required
